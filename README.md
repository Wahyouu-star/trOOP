# TR-OOP 

Proyek ini merupakan tugas akhir OOP berbasis Java dengan konsep MVC dan GUI Java Swing.
Fitur yang ditambahkan pada sistem akademik:
- Dashboard HOME
- Registrasi KST
- Jadwal Kuliah + Absensi
- Transkrip Nilai
- Tagihan
- Notifikasi
- Kalender Akademik
- Multi Role (Mahasiswa / Dosen / Super Admin)
